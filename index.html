What should we do to become a Full-Stack Web Developer?

Here are my agendas:

  1. First, make sure that I'm up for this before starting anything else.
  2. Learn as much HTML as I can.
  3. Learn the best CSS practices and styles to make this website look a lot cooler and cleaner.
  4. Learn JavaScript to give this website life.
  5. Take in other programming languages that will help me become a Full-Stack Web Developer!

How can we achieve our goals? Let's make a check list for all of my agendas.

Let's tackle the first one on the list: Motivation.

  • Put on some music that will not overwhelm our thinking process. Jazz is good for this, not very overpowering, it will keep you awake, keeps you interested, jolly, and it's great for wandering minds.
  • Visualize an idea for a website, this may take some time but exploring sample websites online will speed this up.
  • Find free videos in YouTube, other dev websites, and Google things out to help us with any questions or obstructions with code.
  • Taking a rest if everything is clogging up to keep me from burning-out.

Now for breaking the ice: HTML.

  • Learn the basics, master it, be a carpenter.
  • Invest your time and effort as this skill is very important.
  • Make sure to ask help from usefull websites as we do not have any clue what we're doing at first.
  • Love the lessons you're taking in.
